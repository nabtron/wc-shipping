Removes shipping options from cart total when free shipping threshold is achieved. 

== Description ==
WooCommerce gives an option to set minimum threshold for free shipping, however even when it's received, the paid options don't get hidden themselves. This plugins hides them for you.

Options include turn off or on (by going to settings > WC Shipping.

If useful - Future versions are planned to have more options, so please leave a feedback

Found a bug or have a feature request ? <a href="http://nabtron.com/wc-shipping/">Report here</a>

== Installation ==
1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. goto: settings > WC Shipping and turn it on and you\'re done
